Student Analytics Project
Analyzing Student Performance Using Data Analytics
________________________________________
Overview
This project focuses on analyzing student academic data using a structured data analytics approach.
The workflow includes:
•	Data preprocessing and cleaning 
•	Exploratory Data Analysis (EDA) 
•	Data visualization and insights generation 
The objective is to understand student performance patterns and identify key factors affecting academic success using data-driven methods.
________________________________________
Dataset
Source: Synthetic student dataset (generated using Python)
Description:
The dataset includes features such as:
•	Internal marks 
•	External marks 
•	Attendance percentage 
•	Total marks (calculated) 
________________________________________
Objectives
•	Analyze student data to understand performance trends 
•	Identify key factors affecting student results 
•	Handle missing values and clean the dataset 
•	Generate meaningful insights using visualization 
•	Build a structured and reusable analytics workflow 
________________________________________
Project Highlights
1. Data Preprocessing
•	Generated dataset using NumPy 
•	Handled missing values (Attendance) using median method 
•	Created new column: Total Marks 
•	Ensured data consistency and correctness 
________________________________________
2. Exploratory Data Analysis (EDA)
•	Studied distribution of marks and attendance 
•	Checked relationships between variables 
•	Identified patterns like: 
o	Higher attendance → better performance 
o	Internal marks impact total marks 
________________________________________
3. Visualization
Used charts to understand data clearly:
•	Bar chart → student marks comparison 
•	Histogram → distribution of marks 
•	Pie chart → pass/fail percentage 
•	Scatter plot → attendance vs marks
